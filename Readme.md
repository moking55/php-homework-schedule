## PHP Homework Schedule System
ระบบนี้สร้างขึ้นมาเพื่อเก็บข้อมูลการบ้านและวิเคราะห์เปอร์เซ็นต์การส่งงานรายบุคคล

## ฟีเจอร์

## คำอธิบายโฟเดอร์
- assets มีไว้เก็บข้อมูลที่เกี่ยวกับส่วนหน้าบ้านของผู้ใช้ ภายในจะประกอบไปด้วยไฟล์ .css และ .js เป็นส่วนใหญ่เพื่อรอเรียกใช้จากด้านในเว็บอีกทีหนึง
- core เป็นโฟเดอร์ที่เกี่ยวกับการทำงานหลักในตัวโปรเจกต์ เปรียบได้เหมือนหัวสมองในการคิดและวิเคราะห์ และ Logic จะรวมอยู่ในโฟเดอร์นี้
- uploads โฟเดอร์สำหรับอัพโหลดไฟล์เข้าไปในตัวเว็บไซต์แและจะบันทึกชื่อไฟล์และนำชื่อและที่อยู่ของไฟล์ไปใส่ไว้ในฐานข้อมูล
- views โฟเดอร์ที่เก็บรวบรวมไฟล์ PHP ที่รอเพื่อน้ำมาแสดงผลอีกครั้ง
    - views/components โฟเดอร์ที่จะเอาไว้เก็บไฟล์ที่แยกออกมาเช่น หน้าต้อนรับหน้าแรก หรือส่วนที่เอาไว้แสดงผลเป็นต้น
